# Reneo Studio Booking Website

Reneo Studio Booking Website adalah sistem pemesanan studio foto berbasis web yang dirancang untuk memudahkan pelanggan dalam melakukan reservasi layanan fotografi secara online. Website ini memungkinkan pengguna untuk melihat layanan yang tersedia, melakukan booking jadwal, serta mengelola informasi pemesanan dengan lebih mudah.

## Latar Belakang

Reneo Studio merupakan studio fotografi yang menyediakan berbagai layanan fotografi seperti foto studio, event photography, dan photobooth. Untuk meningkatkan efisiensi pelayanan dan mempermudah pelanggan dalam melakukan reservasi, dikembangkanlah website booking online yang dapat diakses kapan saja melalui internet.

## Fitur Utama

- Melihat informasi layanan fotografi
- Melihat portfolio hasil foto
- Melakukan booking jadwal studio
- Sistem login admin
- Admin dapat mengelola data layanan
- Admin dapat mengelola data booking pelanggan
